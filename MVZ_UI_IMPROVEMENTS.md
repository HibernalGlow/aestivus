# MVZ 节点 UI 改进总结

## 改进内容

### 1. 修复 mvz 源码中的 delete_files 返回码问题
**文件**: `PackU/AutoUnzip/src/mvz/executor.py`

**问题**: 7z 命令返回码为 1（警告）时被认为是失败
**解决**: 修改返回码检查逻辑，将返回码 0 和 1 都认为是成功

```python
# 修改前
if retcode == 0:
    return ExecutionResult(success=True, ...)

# 修改后
if retcode in (0, 1):  # 7z 返回码: 0=成功, 1=警告(但操作可能成功), 2+=错误
    return ExecutionResult(success=True, ...)
```

## 功能说明

### Extract（提取）
- 从压缩包中提取指定文件
- 不删除原文件
- 支持输出目录、就近提取、自动目录、扁平化等选项

### Delete（删除）
- 从压缩包中删除指定文件
- 直接删除，不提取

### Move（移动）
- 从压缩包中提取文件并删除原文件
- 相当于 extract + delete

### Rename（重命名）
- 使用正则表达式重命名压缩包内的文件
